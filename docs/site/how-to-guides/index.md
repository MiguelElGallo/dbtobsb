# How-to guides

Use these guides when you have a specific result in mind:

- [Prepare a dbt project](add-a-dbt-project.md) — create the two source YAML files
  and one named selector required by installation.
- [Install the private release](install-private-release.md)
- [Query observability data](query-observability-data.md)
- [Recover missing evidence](recover-missing-evidence.md)
- [Stop or uninstall dbtobsb](stop-or-uninstall.md)

Before installation, check the [supported environment](../reference/supported-environment.md)
and [security and permissions](../reference/security-and-permissions.md). Query and
recovery guides require access only to the resources named in their first section.
For a guided first experience, start with the
[first-run tutorial](../tutorials/see-your-first-run.md).
