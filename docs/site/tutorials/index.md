# Tutorials

Tutorials help you learn by completing a small, working journey.

Start with one of these:

- [See your first observed dbt run](see-your-first-run.md) — run a prepared observed
  dbt Job and see the captured data.
- [Inspect dbt artifacts locally](inspect-artifacts-locally.md) — use synthetic files
  to see how artifact validation works without connecting to Databricks.

Follow each tutorial in order. If you need to solve one specific problem instead,
use the [how-to guides](../how-to-guides/index.md).
